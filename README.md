# RAMS Design System

> Dieter Rams-inspired design system, website iterations, and UI component experiments

A disciplined exploration of "good design" principles translated into digital interfaces. From a full Braun-aesthetic design system to organic UI experiments — pendulum physics, clock mechanics, pixel art — all sharing the same constraint-driven DNA.

- `design-system.css` — Core CSS variables and tokens
- `RAMS-CSS-PRD.md` — Product requirements doc

### Component Experiments
- `Drams - Framer components inspired by Dieter Rams' principles.html`

## Philosophy
Ten principles. Zero decoration. Function as beauty.
