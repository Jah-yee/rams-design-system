BRAUN
Skeuomorphic icon · 工业设计语言 Web 规范（致敬 Dieter Rams）

设计系统

目标：让网站像“一个硬件整体”——同一气质的面板、按钮、孔阵列与显示屏

原则（Rams-inspired）

P1
克制的拟物（Skeuomorphic-lite）
仅用细边框 + 轻微内高光/内阴影表达结构，不做夸张高光。
P2
功能色彩
橙色用于“可操作/关键功能”，其余保持中性低对比。
P3
结构优先
网格、对齐、留白；组件像“面板上的零件”，逻辑清晰可读。
