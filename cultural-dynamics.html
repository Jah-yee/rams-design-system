Cultural Dynamics

This simulation visualizes cultural dissemination, inspired by the Axelrod model (1997). Each particle represents an agent with discrete cultural traits. Nearby agents interact and may adopt each other's traits based on their similarity. With social repulsion enabled, dissimilar agents actively differentiate themselves, creating cultural polarization.

Simulation Parameters
Agent Speed 0.3x
Day/Night Cycle Speed 1.0x
Traits per feature (Q) 4
